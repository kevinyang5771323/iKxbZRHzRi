# 项目概述 python603

本项目是一款基于Python的 课程信息管理系统，采用Django框架进行开发，使用MySQL作为数据库存储。

## 技术栈

- 后端：Python 3.x
- Web框架：Django
- 数据库：MySQL

## 功能模块

- 课程管理：包括课程的增加、删除、修改和查询等功能。
- 学生管理：对学生信息进行管理，包括增加、删除、修改和查询等操作。
- 成绩管理：对学生的成绩进行管理，包括成绩的录入、修改和查询等。

## 系统角色

- 管理员：负责系统的管理和维护，具有最高权限。
- 教师：可以管理自己所教授的课程和学生信息。
- 学生：可以查看自己的课程和成绩信息。

## 运行环境

- 操作系统：Linux/Windows均可
- Python版本：3.x
- 数据库：MySQL 5.x或更高版本

## 部署步骤

1. 安装Python 3.x环境。
2. 安装MySQL数据库，并进行配置。
3. 安装项目依赖的第三方库（如Django）。
4. 导入项目数据库。
5. 运行项目。

## 目录结构

```
project/
│
├── app/              # 应用目录
│   ├── migrations/   # 数据库迁移文件
│   ├── models.py     # 模型定义
│   ├── views.py      # 视图处理逻辑
│   └── admin.py      # 管理后台配置
│
├── project/          # 项目配置目录
│   ├── settings.py   # 项目配置文件
│   ├── urls.py       # URL配置文件
│   └── wsgi.py       # WSGI配置文件
│
└── manage.py         # 项目管理脚本
```
